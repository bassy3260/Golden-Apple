# Golden-Apple
 인공지능을 활용한 농산물 구매 도움 서비스
## 개요
+ 경상국립대학교(GNU) CS22 종합설계PBL(졸업 프로젝트) 작품 


## 팀원
+ 박미성: 팀장,농산물 품질 판별 A.I 개발
+ 백대홍: 농산물 시세 예측 A.I 개발
+ 김태현: 안드로이드 프론트엔드, 거주지 주변 농산물 직판매장 검색 기능 구현
+ 원서영: 안드로이드 프론트/백엔드, 커뮤니티 기능 구현
